1. Какая актуальная версия HTML, CSS, javascript?
    Ответ: HTML5, CSS3, Javascript ES10.
  2. Классифицируйте теги (приведите минимум две классификации).
    Ответ: структурные, блочные, строчные, строчно-блочные встраиваемые.
  3. Что такое блочный элемент?
    Ответ: предназначен для создания структуры основных частей html-страницы, для разделения контента на логически связанные блоки, занимает всю доступную ширину.
  4. Что такое строчный элемент?
    Ответ: является частью строки, используется для выделения текста и изменения его вида? нельзя задавать ширину и высоту.
  5. Допустимо ли вставлять блочный элемент в строчный?
    Ответ: можно вставлять строчный элемент в строчный, блочный элемент в строчный вставлять запрещено.
  6. Как из квадрата сделать круг?
    Ответ: через border-radius: 50%;
  7. Как отобразить сектор круга?
    Ответ: задать border-radius, border и сторону, в которой хотите выделить сектор круга. Например: div { width: 0px; height: 0px; border-radius: 50%; border: 50px solid black; border-top: 50px solid red; }
  8. Чем контейнер отличается от тега?
    Ответ: контейнер - это парные теги, т.е. состоят из открывающего и закрывающего тега.
  9. Почему следует использовать спрайты?
    Ответ: для сокращения запросов на сервер, соотвественно меньше нагрузка и быстрее работает сайт.
  10. Перечислите способы сделать элемент невидимым на странице.
    Ответ: overflow: hidden, visibility: hidden, display: none, opacity: 0
  11. Какое позиционирование у контейнера html?
    Ответ: изначально позиционирование, если оно не задано, имеет значение static.
  12. Перечислите способы назначения стилей.
    Ответ: через link, тег style, внутри тегов (например div style="width: 100px" /div)
  13. Почему у родителей элементов с position: absolute, float:left (при отсутствии другого контента) высота становится 0?
    Ответ: они выпадают из общего потока html-страницы.
  14. Перечислите все способы (какие знаете) отобразить div в строку.
    Ответ: display:inline-block, через float, display: flex, display: grid, display: table-cell, через position.
  15. Почему у body по-умолчанию отступы слева-справа, сверху-снизу? Как их убрать?
    Ответ: убрать их можно с помощью margin: 0,
  16. Какие типы селекторов вы знаете? (перечислить)
    Ответ: селекторы по классам, тегам, id и универсальные селекторы
  17. Для чего нужны псевдоэлементы?
    Ответ: они позволяют задать стиль элементам, которых нет в структуре документа, а также показать содержимое, которого нет в исходном коде.
  18. Для чего нужны псевдоклассы?
    Ответ: они дают возможность стилизовать определенные элементы, определяет его состояние (например зависящее от пользователя, как active, hover или checked) или положение в html.
  19. Перечислите атрибуты тега img.
    Ответ: src, alt, width, height, title.
  20. Сколько запросов одновременно может отправить браузер на получение изображений/стилей/файлов?
    Ответ: не имеет ограничений.
  21. Для чего нужно кеширование?
    Ответ: оно позволяет увеличить производительность сайтов, запросов и пр. за счет использования ранее сохраненных данных (например поиск записей в googlу, которые вы уже искали), соответственно сервер работает быствее за счет этого.
  22. С помощью какого css-свойства можно изменить вид курсора?
    Ответ: с помощью cursor.
  23. С помощью какого свойства подключаются свои шрифты в css?
    Ответ: с помощью @import, тега link и @font-face
  24. Чем отличается min-width от width? Что будет если min-width > width?
    Ответ: width - устанавливает ширину элементу, min-width - устанавливает минимальную ширину элементу (например: при заданной min-width: 600px и уменьшении ширины экрана меньше 600px? то внизу появится горизонтальный scroll). Если min-width > width, то ширина будет min-width.
  25. Что такое кросс-браузерность?
    Ответ: одинаковое отображение и работы страниц / сайтов в разных браузерах.
  26. На какой вкладке dev-tool можно имитировать медленный интернет?
    Ответ: через network
  27. Что такое :hover?
    Ответ: отображение прописанного стиля для элемента при наведении на него курсора мыши.
  28. "Прочитайте" селектор как браузер: body>div>p:first-child:not(.store):not(#message)
    Ответ: применять для параграфа первого ребенка, что не является классом store с идентификатором message, который вложен в div, который вложен в body
  29. Какие плюсы в использовании иконочных шрифтов?
    Ответ: их можно увеличивать без потери качества, большая скорость загрузки, т.к. небольшой размер, доступны стили css.
  30. Чем em, отличается от ex?
    Ответ: em - относительные ед. измерения, определяются по тексту на данный момент. ex - высота символа «x» в нижнем регистре (загуглила)). Для них примениются одинаковые правила (они привязаны к размеру шрифта по умолчанию или к размеру шрифта родительского элемента).
  31. Что такое rem?
    Ответ: Единица измерения, которая соответствует font-size корневого элемента html (по умолчанию 1rem=16px).
  32. Где лучше располагать в html коде css?
    Ответ: лучше распологать в отдельном теге style или в отдельной файле и ссылкой на него link rel="stylesheet" href=""
  33. Как избежать "прыжков" контента при загрузке изображений?
    Ответ: изначально указывать ширину и высоту изображений.
  34. Что происходит, если указать только один из габаритов у изображения?
    Ответ: изображение будет уменьшено или увеличино (в зависимости, что будет указано), пропорционально указанной единице измерения.
  35. Как инициировать git-репозиторий?
    Ответ: через git init? git add ., git commit -m ''
  36. Как добавить файл на отслеживание?
    Ответ: git add (имя файла), git add .
  37. Как создать контрольную точку git?
    Ответ: git commit -m ''
  38. Как отменить изменения в файле до последней контрольной точки в ветке?
    Ответ: через git checkout
  39. Чем git pull отличается от git push? А от git clone?
    Ответ: git pull - получения данных из удаленного репозитория на текущий (на котором работаете). git push - отправляет данные на удаленный репозиторий. git clone - создание копии репозитория, например в другую папку.
  40. Что будет, если задать padding-left: -40px;
    Ответ: у padding нет отридцательных значений.
  41. Для чего использовать свойство title у HTML-элемента?
    Ответ: для доп. информации в html-элементе в виде всплывающей подсказки (подсказки).
  42. Как назначить фон третьему элементу в контейнере body?
    Ответ: например, body>div:nth-child(3) {background: red;}, т.е. через псевдокласс, также можно через стиль или класс для элемента.
  43. Как сделать изображение на странице полупрозрачным?
    Ответ: через opacity: 0.5.
